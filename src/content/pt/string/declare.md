---
title: Declarar strings
---

Você pode declarar uma string (uma sequência de caracteres) usando:

-   `"`: aspas simples
-   `'`: aspas dupla
-   `` ` ``: crase

```javascript
// `s1`e`s2` são equivalentes
const s1 = "Aspas";
const s2 = 'Aspas simples';

// String com crases permite textos de várias linhas
const s3 = `Crase
linha1
linha2`;
```
